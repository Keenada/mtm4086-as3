Node Timers
Use a timer or you'll blow your node
Timer is a global module and is included by default within a Node.js project. Timers are used to time executions of code after, or repeating after, a set time in milliseconds.

setTimeout(callback , delay[,args][,...])

setTimeout is used for to execute a single callback after a specified delay in milliseconds. Callback and delay are required arguments and arguments can be pass to the callback.

*Important note: Node.js doesn't guarantee the execution of callback function when the milliseconds from delay are met.

Improtant note: Node.js doesn't guarantee the execution of callback function when the milliseconds from delay are met.

setTimeOut(function(str1, str2){
    console.log(str1 + “ ” + str2);
}, 1000, “Hello”, “World”);

This example with print “Hello World” to the console after 1000 milliseconds. SetTimeout returns a timeoutObject for possible use with clearTimer()

clearTimer(timeoutObject)

This function will prevent a timer from executing another time and release the object from memory.

var timer = setTimeOut(function(str1, str2){
	console.log(str1 + “ ” + str2);			
}, 1000, “Hello”, “World”);
clearTimer(timer);

This example with print “Hello World” to the console after 1000 milliseconds. Than releases the timer.

setInterval(callback , delay[,args][,...])

Functions similarly to setTimeout except that it executes the callback every delay milliseconds. You can pass arguments the same way as for the callback.

Improtant note: Actual delay is less than given delay.

setInterval() also returns an intervalObject for possible use in clearInterval() later.

var i = 0;	
setInterval(function(i){
    console.log(i);
     i++;
			
}, 1000, i);

This code will print i to the console then add 1 to i. Repeat ad nauseum.

clearInterval(intervalObject)

setInterval() also returns an intervalObject for possible use in clearInterval() later.

var i = 0;			
var timer2 = setInterval(function(i){
	console.log(i);
	i++;			
}, 1000, i);
				
$(“button”).on('click', function(){
	clearInterval(timer2);	
});

This code will print i to the console then add 1 to i. It will repeat until button is pressed.
