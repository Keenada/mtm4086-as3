What is PhoneGap?

PhoneGap is an open source solution for building cross-platform mobile apps with standard web languages like HTML, javaScript, and css. It is a mobile application development framework that is based upon the open source Apache Cordova project.

Using the PhoneGap API you are able to access multiple of the features of a mobile device. A couple of these are Geolocation, Camera, Notifications(Vibrate, Sound, Alert), and many more. You can view more about this plugins at this link if you are interested. For this example, we are going to simply talk about how to turn your web files into a mobile application, specifically android.

Steps for building an android application on a windows platform

  • Install NODEJS - Click Here
  • Install Android SDK Manager - Click Here
  • Install Ant - Click Here
  • Open up your command prompt and install phonegap
    • npm install -g phonegap
  • Create a directory to hold the files
    • phonegap create test com.anything.test Test
    • "test" is the name of the directory you are about to create
    • "com.anything.test" is a reverse domain (basically a unique id)
    • "anything" is usually the domain of the app you are building
  • Change into the new directory you just created
    • cd test
  • In the file explorer go into the "www" folder and add your source files(html, css, js)
  • Set the android path pt 1
    • set ANDROID_HOME=(your android sdk location)
    • For example, my path was - set ANDROID_HOME=C:\Users\Nick\AppData\Local\Android\sdk
  • Set the android path pt 2
    • set PATH=%PATH%;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools
  • Add the mobile application platforms
    • phonegap platform add android
  • set the ant path pt 1
    • set ANT_HOME=(your ant location)
    • For example, my path was - set ANT_HOME=C:\ant
  • Set the ant path pt 2
    • set PATH=%PATH%;C:\ant\bin
  • Build the application
    • phonegap build android
  • View the app through your SDK
    • phonegap emulate android
  • Send the application to your mobile device
    • phonegap serve
    • Install the phonegap developer application on your android device and match the connection. This allows you to see changes live on your mobile device
