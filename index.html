HOTP

HOTP generates a one-time password using a counter-based algorithm. When generating the password you can specify the key, counter, password length, as well as the encoding (ASCII, hexadecimal, or base32). The generated password will be returned as a string.

Parameters

  • Key: the secret key in ascii, hex, or base32
  • Counter: the moving factor
  • Length(default 6): length of generated password
  • Encoding(default ascii): the encoding of the key

Code Sample

speakeasy.hotp({key: 'kitten', counter: 582, length: 8});

Output

27808785

TOTP

TOTP generates a one-time password using a time-based algorithm. When generating the password you may specify the key, step, time, intitial time, length, and the encoding.

Parameters

  • Key: the secret key in ascii, hex, or base32
  • Step(default 30): the time step, in seconds
  • Time (default current time): the time to calculate the TOTP from
  • Initial_time (default 0): the starting time to calculate the TOTP from
  • Length(default 6): length of generated password
  • Encoding(default ascii): the encoding of the key

Code Sample

speakeasy.totp({key: 'kitten', step: 60});

Output

655306

Generate Key

Generates a random secret key in ascii, hex, or base32. While generating the key you can specify the length, symbols, qr codes, google authenticator qr codes, and name.

Parameters

  • Qr_codes (default false): generate links to qr codes for each encoded key
  • Symbols (default true): include symbols in the key
  • Google_auth_qr (default false): generate a link to a qr code
  • Name(optional): give a name if you used googe_auth_qr
  • Length(default 32): length of generated password

Code Sample

speakeasy.generate_key({length: 15, qr_codes: true});
