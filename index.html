What is Node?

Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. It is an open source, asynchronous framework that is used for developing web applications.

To use the console module, you have to install Node.js and then check if it got installed properly by typing in node -v into your command processor to check for the installed version number.

What is Console Module?

Console module is a debugging console where it is used often as Node’s output functionality to display messages and errors.

It is similar to the Javascript console log that people are familiar with in the web browsers. While the API for the console class in Node.js is similar to the web browser, it was not intended to duplicate the browser’s functionality.

Right away after installing Node.js, the console module is available to us globally and we can use this to log messages into the console. Since node.js is working on the server side, it will write to stdout and stderr. stdout is the file where normal output messages are stored and stderr is the file where error messages are stored.

Node.js Example

In an empty .js file, type in
console.log("Hello Everyone");
This string, "Hello Everyone" will show up in the Terminal if you call on it with node and then the script file name
node node_console

Web Browser Example

In an empty .js file, type in
console.log("For example...");
Activate the debugger software on the web browser and navigate to the Console tab to see the results

Console Module Components

The console module exports two different components:

  1. A console class such as console.log( ), console.error( ), and console.warn( ) that can be written to any Node.js stream
  2. A global instance will write to stdout, stderr and can be used without attaching the require ‘console’

Console Class Example

Define the variables.
const out = getStreamSomehow();
const err = getStreamSomehow();
const myConsole = new console.Console(out, err);

This will print "Hello Everyone" to out
myConsole.log('Hello Everyone');

This will print [Error: This is not allowed], to err
myConsole.error(new Error('This is not allowed'));

Global Instance Example

This will print "Hello Everyone" to stdout
console.log('Hello Everyone');

This will print [Error: This is not allowed], to stderr
console.error(new Error('This is not allowed'));

Console Module Classes

There are two methods to access console classes that can be written to Node.js output streams.

const Console = require('console').Console;
const Console = console.Console;
