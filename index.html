Papercut for Node

Researched by: Alain Filion

Papercut is a module that claims to crop and resize images.

This two year old module which has not been updated since. I was unable after much trying to get any part of the module working.

WARNING: Stay away from old modules that have not been modified in over a year. They probably won't work with the up to date node.
If ever you need get the error in teminal:
"Please check if you have git installed and in your PATH"

1. Search in control panel for 'environment variables' --This will come up under your Advanced System Settings. Click on the button with the same title.

2. Scroll down the list of System Variables until you see the variable titled 'Path' --Select it and click edit.

3. If you are on windows copy this path onto the front of the list:

C:\Program Files (x86)\Git\bin;

***Just in case you were wondering -- Do Not Delete Anything***
