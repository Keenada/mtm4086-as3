Welcome to Faker

About

The Faker node is a module that allows users to get fake information generated to them with the click to a button. It includes a wide variety of information, for example:

  • Name
  • Address
  • Internet
  • Phone Number
  • Image
  • Company
  • Lorem Ipsum
  • Helpers
  • Date

Plus all of these have sub info so if you wanted just a first name field and a last name field you would use firstName and lastName. Another example is you can go into more details about your address by getting a zip code, city, street, province and country.

Uses

People could use this for when they are creating fake users (ex: user personas) and need information that looks and sounds real. You can also use it to create a fake identity to get into those R rated movies.

Installation

Faker can be installed using NPM, just like most Node modules.

$ Npm install faker
